# PsiLab · Third Eye (working title)

Next.js 15 starter scaffold for a playful daily-practice app that tests intuition drills with honest statistics.

## Stack

- Next.js 15 (App Router) + TypeScript + Tailwind CSS
- Neon Postgres + Drizzle ORM

## Available scripts

```bash
npm run dev
npm run lint
npm run build
```

## Current scope

This repository currently includes:

- Dark-mode layout shell and visual direction mockups for the home screen
- Three mascot options with deadpan sample lines
- Initial Drizzle schema sketch based on the product primer

Functional drill/gameplay flows are not implemented yet.
